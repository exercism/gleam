# Design

## Learning objectives

- Know what a bit string is.
- Know how to create a bit string literal.
- Know about the `:` binary size modifier.
- Know how to construct a bit string.
- Know how to pattern match a bit string.

- Erlang FFI.
- JavaScript FFI.
- Comparison with strings.
